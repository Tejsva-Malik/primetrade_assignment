# Bitcoin Market Sentiment vs Trader Performance Analysis

## Objective
This project analyzes the relationship between Bitcoin market sentiment (Fear/Greed Index) and trader performance using historical trading data.

## Datasets Used
1. Bitcoin Fear & Greed Index Dataset
2. Historical Trader Data from Hyperliquid

## Tools & Technologies
- Python 3.11
- Pandas
- Matplotlib
- PyCharm

## Analysis Performed
- Data loading and cleaning
- Date conversion
- Dataset merging
- Sentiment-based profit analysis
- Data visualization using bar charts

## Key Insights
- Trader profitability changes according to market sentiment.
- Average Closed PnL varies during Fear and Greed periods.
- Market sentiment influences trading behavior.
